Установть node.js 7.9 отсюда https://nodejs.org/en/
Открываем терминал в каталоге со скриптами и набираем
npm install
Выполнить нужно 1 раз

Дале для запуска скриптов
вместо ид_дгруппы_донера могут быть названия из url

Парсинг группы:
node wallget.js ид_дгруппы_донера

Парсинг фото
node photos.js ид_дгруппы_донера_или_профиля

Постинг постов из файла
Данные берутся из файла ид_дгруппы_донера.xlsx
node post.js ид_дгруппы_донера ид_дгруппы_куда
